Wiggle — Back Home Swipe Spec

One swipe picks up. Second swipe sends home. Undo lives in the profile.

The Rule (WWRS Filter)
👆Swipe left once → Picked up. Timestamp logged. Card turns sage green.
👆Swipe left again → Back home. Timestamp logged. Card fades out.
🐾Tap dog name → Opens profile. All detail lives there.
✏️Undo = edit timestamps in profile. No separate undo button on the card.
📦Mini card shows only: #, dot, name, address, door code. Nothing else.
